# 로또 번호 추천 사이트

## 개요

이 애플리케이션은 사용자에게 로또 번호를 추천해주는 간단한 웹 애플리케이션입니다. 사용자가 버튼을 클릭하면 1부터 45까지의 숫자 중 6개의 숫자를 무작위로 선택하여 화면에 표시합니다. 또한, 사용자는 다크 모드와 라이트 모드 사이를 전환하여 애플리케이션의 시각적 테마를 변경할 수 있습니다.

## 디자인 및 기능

*   **헤더:** "Lotto Number Generator"라는 제목을 표시합니다.
*   **테마 토글 버튼:** "Toggle Theme" 버튼을 클릭하여 다크 모드와 라이트 모드 사이를 전환합니다. 사용자의 테마 선호도는 브라우저의 로컬 스토리지에 저장되어 다음에 방문할 때 자동으로 적용됩니다.
*   **버튼:** "행운의 번호 받기" 버튼을 클릭하면 새로운 로또 번호를 생성합니다.
*   **번호 표시:** 생성된 6개의 로또 번호를 시각적으로 강조하여 표시합니다. 각 번호는 동그란 원 안에 표시되며, 번호별로 다른 색상을 가집니다.
*   **반응형 디자인:** 모든 화면 크기에서 잘 보이도록 반응형으로 디자인합니다.
*   **다크/라이트 모드:** 사용자의 선호에 따라 애플리케이션의 색상 테마가 변경됩니다.

## 현재 구현 계획 (완료)

1.  **HTML (`index.html`):** 기본 구조를 설정합니다.
    *   제목, 테마 토글 버튼, 로또 번호 생성 버튼, 번호가 표시될 컨테이너를 포함합니다.
2.  **CSS (`style.css`):** 사이트의 디자인과 다크/라이트 모드 테마를 적용합니다.
    *   색상, 글꼴, 레이아웃 등 시각적 요소를 정의하고, CSS 변수를 사용하여 테마를 관리합니다. `.dark-mode` 클래스를 통해 다크 모드 스타일을 정의합니다.
3.  **JavaScript (`main.js`):** 로또 번호 생성 및 표시 기능과 테마 전환 기능을 구현합니다.
    *   버튼 클릭 이벤트를 처리하고, 중복되지 않는 6개의 랜덤 숫자를 생성하여 화면에 표시합니다.
    *   테마 토글 버튼 이벤트를 처리하고, `body` 요소에 `dark-mode` 클래스를 토글하며, 사용자의 테마 선호도를 `localStorage`에 저장합니다. 페이지 로드 시 저장된 테마를 적용합니다.
